# Handoff: Лендинг «История Муминат» (курс «Воспитание детей в Исламе 5.0»)

## Overview
Одностраничный вертикальный scrollytelling-лендинг — реальная история ученицы курса
(Муминат) от «до» к «после», завершающийся призывом записаться на новый поток курса
Солихи Умм Мусы. Цель — эмоционально вовлечь маму-мусульманку и привести к CTA
«Записаться на курс» с промокодом и таймером обратного отсчёта.

## About the Design Files
Файлы в этом пакете — **дизайн-референс, созданный в HTML** (прототип, показывающий
задуманный внешний вид и поведение), а не готовый продакшн-код для копирования.
Задача — **воссоздать этот дизайн в среде вашего кодовой базы** (React, Vue, Next.js,
Astro и т.п.) по её принятым паттернам и библиотекам. Если среды ещё нет — выберите
подходящий фреймворк и реализуйте дизайн в нём. Исходник — единый Design Component
(`.dc.html`), использующий кастомный рантайм проекта; в реальном проекте это должен быть
обычный компонент/страница.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, отступы, анимации и копирайт.
Воссоздавать пиксель-в-пиксель, подставляя ваши библиотеки/утилиты для стилей.

## Design language (кратко)
Премиальный, спокойный, «ночное небо → рассвет». Тёмно-синие звёздные секции сменяются
тёплыми кремовыми. Единственный акцент — матовое золото. Заголовки — Playfair Display
(часто одна фраза курсивом с золотым градиентом), текст/UI — Montserrat, скриптовое
имя автора/подпись — Hamiltone. Орнаменты: золотой ромб (повёрнутый на 45° квадрат)
и тонкие золотые линии-градиенты.

## Design Tokens

### Цвета
- Фон-небо (тёмный, фикс): радиальные точки-звёзды + `radial-gradient(circle at 50% 30%, #16223e 0%, #0c1426 55%, #070b16 100%)`
- Тёмный текст-основа: `#F6F1E7`; приглушённый body-текст на тёмном: `#DAD3C6`; серифный светлый: `#EFE7D5`
- Светлые секции (фон): `#F2EBDD`, `#FAF6EE`, `#efe6d3` (чередуются для ритма)
- Текст на светлом: заголовки `#1B2A3A`, body `#4a4436`, приглушённый `#5c5340`
- Золото (основной акцент): `#C7A86B`; тёмное золото (на светлом фоне): `#A77E3F`
- Золотой градиент (текст/кнопки): `linear-gradient(135deg,#E3D2A8,#C7A86B 48%,#A77E3F)`
  (для скрипт-имени: `linear-gradient(135deg,#F0E4C4,#E3D2A8 30%,#C7A86B 62%,#A77E3F)`)
- Зачёркивание «было» (before/after): `#8a8272` текст, линия `#c98a8a`
- «Тревожный» акцент (дедлайн/сгорание): `#d9a9a9`
- Карточки таймера (фон): `linear-gradient(150deg,#12203a,#0a1220)`

### Типографика
- Дисплей-заголовки: **Playfair Display**, weight 500–800; крупные — 800; курсивная золотая фраза — italic 800
- Текст/UI/надзаголовки: **Montserrat**, weight 300 (body), 400/500, 600 (eyebrow/пиллы)
- Скрипт (только имя автора и подпись `@detisestram`): **Hamiltone** (кастомный OTF, `fonts/Hamiltone.otf`)
- Eyebrow-метки: Montserrat 600, `letter-spacing:.26em`, UPPERCASE, золото; размер `clamp(.72rem,1.4vw,.84rem)`
- H1 герой: `clamp(2.8rem,8vw,5.8rem)`, line-height 1.05
- H2: `clamp(2rem,4.6vw,4rem)`
- Body: `clamp(1.05rem,1.8–1.9vw,1.28–1.34rem)`, line-height 1.7–1.75, max-width 44–52ch
- ⚠️ Для текста с `background-clip:text` (золотой градиент) добавлять `padding-bottom:.08em`;
  для Hamiltone-скрипта также запас сверху (`line-height:1.7; padding:.28em .08em .18em`) —
  иначе срезаются росчерки букв.

### Отступы, радиусы, тени
- Секции: `padding: 9–12vh 8vw`; внутренний контейнер `max-width:1080px; margin:0 auto`
- Полноэкранные секции: `min-height: 70–96vh`, flex-центрирование
- Радиусы: карточки модулей 20px, before/after-строки 16px, карточки таймера 14px, кнопка/пиллы `999px`
- Тень карточек (на светлом): `0 12px 32px rgba(27,42,58,.08)`, бордер `1px solid rgba(199,168,107,.4)`
- Кнопка CTA: тень `0 14px 38px -8px rgba(199,168,107,.55), inset 0 1px 0 rgba(255,255,255,.35)`

## Interactions & Behavior
- **Scroll-reveal:** каждый помеченный элемент стартует `opacity:0; transform:translateY(36px)`,
  при попадании во вьюпорт (IntersectionObserver, threshold .15, `rootMargin:0px 0px -6% 0px`)
  переходит в `opacity:1; transform:none`. Переход `opacity .9s / transform 1s cubic-bezier(.16,1,.3,1)`,
  внутри секции — каскад через `transition-delay` (.08–.42s).
- **Звёздное небо:** ~72 точки-`div` (2px, ~22% золотые `#C7A86B`, остальные `#d8dcf0`) со случайной
  позицией/задержкой, анимация мерцания `mmTwinkle` (opacity/scale, 3–6s, бесконечно).
- **Золотой glow (секции 8 и 15):** radial-gradient золотом, `filter:blur`, анимация `mmPulse`
  (opacity + scale, 6–7s).
- **Таймер:** отсчёт до 17 июля 23:59 (локальное время), обновление раз в секунду; при
  истечении показывает нули. Формат: дни без ведущего нуля, часы/минуты/секунды с ведущим нулём.
- **Кнопка CTA:** ссылка на `https://telegram.me/kurs_po_vospitaniyu` (target=_blank).
- **Подпись @detisestram:** ссылка на `https://t.me/detisestram`.
- `prefers-reduced-motion: reduce` — все reveal показываются сразу, мерцание/пульс отключены.
- **Ссылки по умолчанию:** `a{color:#C7A86B} a:hover{color:#E3D2A8}`.

## State Management
Минимально: локальный интервал таймера (setInterval, очищать при размонтировании) и
IntersectionObserver для reveal. Данных с бэкенда нет — весь контент статичный.
